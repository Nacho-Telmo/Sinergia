## 📦 Instalación en Arch Linux (AUR)

Podés instalar **Sinergia DD Burner** fácilmente utilizando tu helper de AUR favorito (como `yay` o `paru`).

[![AUR version](https://img.shields.io/aur/version/sinergia-dd-burner?color=blue&logo=arch-linux)](https://aur.archlinux.org/packages/sinergia-dd-burner)

```bash
# Con yay
yay -S sinergia-dd-burner
```

```bash
# Con paru
paru -S sinergia-dd-burner
```
## 📦Instalacion en Debian y derivadas

![GitHub release](https://img.shields.io/github/v/release/Nacho-Telmo/Sinergia)

1   Descarga el archivo .deb de la última versión en la sección de Releases.

2   Abre tu terminal en la carpeta Descargas y ejecuta el siguiente comando: 

```bash
sudo apt install ~/Descargas/sinergia-dd-burner_*.deb
```
